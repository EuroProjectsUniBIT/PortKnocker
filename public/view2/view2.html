This is a simple Port Knocker that works with a .rsc filter configuration.
The knocker makes only TCP and UDP requests from port 1 to 65535.
The Port Knocker only works with firewall configuration generated by the Filter Generator.
After generating a .rsc file with a firewall filter configuration you can open it with the Port Knocker.
